<strong>下面看一段鸡汤的熬制过程：</strong>

如果把字母 a 计为 1、b 计为 2、c 计为 3 …… z 计为 26， 那么`knowledge = 96` `hardwork = 98` `attitude = 100`,

结论就是：知识（knowledge）与勤奋（hardwork）固然都很重要；但是，决定成败的却是态度（attitude）！

怎么样，这样的论证过程是不是很容易让人误会。鸡汤好喝，可也不是这样熬的啊！

<strong>下面几行代码就来论证这碗鸡汤熬制过程中的 bug：</strong>

```py
import os  # 0.引入第三方python库，这里面的功能拿来就用，不需要自己写

if os.path.isdir('E:\\englishlist'):  # 1.先判断 电脑E盘 有没有englishlist这个文件夹
    pass
else:
    os.mkdir('E:\\englishlist')

# 5. 这个函数的作用是 计算出  单词的值，最终得到 a 计为 1、b 计为 2、c 计为 3 …… z 计为 26
def sum_of_word(word):
    sum = 0
    for char in word:
        sum += ord(char) - 96
    return sum

# 2. 创建一个 result.txt 文本文件
with open('E:\\englishlist\\result.txt','w') as result:
    # 3. 读取 words_alpha.txt这个文本文件，
    with open('D:\\项目参考\\suimu2020\\00Python\\1.5.7文件\\words_alpha.txt','r') as file:
        # 4. 循环遍历 file.readlines() 这个 list 列表
        for word in file.readlines():
            # 6.将某个单词内各字母相加等于100 的单词保存下来
            if sum_of_word(word.strip()) == 100:
                print(word)
                # 7.将这些符合要求的单词保存到 result.txt文件里面
                result.write(word)
```

result.txt 文件夹里面获取到了符合条件的单词。在几秒钟内从 370,099 个英文单词中找到 3,771 个如此计算等于 100 的词汇。

其中等于 100 的词汇有 3000 多个，我随便举几个例子 `abstrusenesses`、 `acupuncturist`、 `adenochondrosarcoma`、 `adenomyxosarcoma`

你知道这个 result.tx 文件里面还有什么吗? 还有这些词 `connivance（纵容）`、`impotence（阳痿）`、 `stress（压力）` 、`flurry（慌张）` 不过怎么看都不像是 100 分的正面词汇。

几行代码就揭示了这个论证过程的荒谬。明明鸡汤好喝，却给这样的熬制过程毁了。
